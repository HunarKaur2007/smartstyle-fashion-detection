# SmartStyle – Real-Time Fashion Detection

A computer vision project for detecting clothing items and enabling virtual try-on using deep learning.

## Technologies
- Python
- PyTorch
- YOLOv8
- ResNet50
- OpenCV

## Features
- Real-time clothing detection
- Similar item search
- Virtual try-on concept

## Project Overview
The system uses YOLOv8 for object detection and ResNet50 for feature extraction to identify clothing items and recommend similar products.
